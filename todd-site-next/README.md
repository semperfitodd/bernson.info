# Todd Bernson - Next.js Site

SEO-optimized static site for Todd Bernson, Chief AI & Technical Officer.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Static output in `out/` directory.

## Deploy

Upload `out/` directory to S3 or any static hosting service.

## Articles

Place `articles.json` in `public/` directory with article data.
